# Display Modes

BitBonsai offers two interface modes to match your workflow.

## Zen Mode (Default)

The clean, focused interface. Perfect for most users.

**What you see:**
- Simple queue with status and progress
- Basic filters (status, search)
- Essential actions (pause, resume, cancel)

**What's hidden:**
- Bulk operations
- Node management details
- Technical debug information
- Advanced filters

<Check>
**Recommendation:** Start with Zen Mode. If you need more control, switch to Advanced Mode.
</Check>

## Advanced Mode

Full power-user interface with all controls visible.

**Additional features:**
- Bulk actions (retry all, cancel all, rebalance)
- Node status bar with health/capacity
- Library and node filters
- Policy and node columns
- Technical encoding metrics
- Debug information panels

### Enable Advanced Mode

<Steps>
  <Step title="Open Settings">
    Click the gear icon in the sidebar
  </Step>
  <Step title="Find Advanced Mode">
    In the Advanced Settings tab, find "Advanced Mode"
  </Step>
  <Step title="Toggle On">
    Switch the toggle to enable power-user controls
  </Step>
</Steps>

<Info>
Your preference is saved and persists across sessions.
</Info>

## Comparison

| Feature | Zen Mode | Advanced Mode |
|---------|----------|---------------|
| Queue view | Simple | Detailed |
| Bulk actions | Hidden | Visible |
| Node status | Hidden | Visible |
| Debug info | Hidden | Visible |
| Learning curve | None | Moderate |

## When to Use Each

<AccordionGroup>
  <Accordion title="Use Zen Mode when...">
    - You're new to BitBonsai
    - You have a single-node setup
    - You just want videos encoded without fuss
    - You prefer clean, minimal interfaces
  </Accordion>

  <Accordion title="Use Advanced Mode when...">
    - You're running multi-node clusters
    - You need bulk job management
    - You're troubleshooting encoding issues
    - You want detailed progress metrics
  </Accordion>
</AccordionGroup>
