## License-API Integration

Website fetches live pricing data from the license-api backend.

<Tabs>
  <Tab title="Development">
    ```typescript
    apiUrl: 'http://localhost:3200/api'
    ```
  </Tab>
  <Tab title="Production">
    ```typescript
    apiUrl: 'https://api.bitbonsai.io/api'
    ```
  </Tab>
</Tabs>

### Pricing Endpoint

**GET /api/pricing** - Public endpoint (no auth required)

<CodeGroup>
```typescript Response
PricingTier[] = [
  {
    id: "cm3abc123",
    name: "FREE",
    displayName: "Free",
    description: "Perfect for hobbyists",
    maxNodes: 1,
    maxConcurrentJobs: 2,
    priceMonthly: 0,
    priceYearly: null,
    isActive: true
  }
  // ... more tiers
]
```

```typescript Service Usage
import { PricingApiService } from '@services/pricing-api.service';

export class PricingComponent {
  tiers$ = this.pricingApi.getActiveTiers();

  constructor(private pricingApi: PricingApiService) {}
}
```
</CodeGroup>

<Note>
API automatically converts prices from cents to dollars for display.
</Note>

## Development

<Steps>
  <Step title="Start Dev Server">
    ```bash
    nx serve website
    ```
    Access at **http://localhost:4201**
  </Step>

  <Step title="Environment Setup">
    Create `src/environments/environment.ts`:
    ```typescript
    export const environment = {
      production: false,
      apiUrl: 'http://localhost:3200/api',
      licenseApiUrl: 'http://localhost:3200/api'
    };
    ```
  </Step>

  <Step title="Watch Mode">
    ```bash
    nx serve website --open
    ```
    Opens browser automatically with live reload
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="API Connection Errors" icon="triangle-exclamation">
    **Symptom:** "Failed to load pricing data"

    **Fix:**
    1. Check license-api is running: `nx serve license-api`
    2. Verify API URL in `environment.ts`
    3. Check CORS settings in license-api
  </Accordion>

  <Accordion title="Build Failures" icon="circle-xmark">
    **Symptom:** `nx build website` fails

    **Fix:**
    ```bash
    # Clear cache
    nx reset

    # Reinstall dependencies
    rm -rf node_modules package-lock.json
    npm install

    # Rebuild
    nx build website
    ```
  </Accordion>

  <Accordion title="Favicon Not Showing" icon="image">
    **Symptom:** Browser shows default favicon

    **Fix:**
    1. Verify `public/favicon.ico` exists
    2. Clear browser cache (Ctrl+Shift+R)
    3. Check `index.html` links:
    ```html
    <link rel="icon" type="image/x-icon" href="favicon.ico">
    <link rel="icon" type="image/svg+xml" href="favicon.svg">
    ```
  </Accordion>
</AccordionGroup>
