Nodes are the workers that handle all your video encoding. They can be your local machine, a dedicated server, or multiple computers working together.

Getting Started:

  1. Click "Register Your First Node" below
  2. A registration command will appear - copy it
  3. Open a terminal on the machine you want to use as a node
  4. Paste and run the command to install the node software
  5. Enter the 6-digit pairing code shown on your node into BitBonsai
  6. Your node will appear here and start accepting encoding jobs!

Tip: Nodes with GPU acceleration (NVIDIA, AMD, Apple Silicon) encode much faster!
